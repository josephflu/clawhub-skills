# EagerBots Skills for ClawHub

A collection of OpenClaw skills published by [EagerBots](https://eagerbots.ai) on [ClawHub](https://clawhub.ai).

Each skill is a self-contained directory with a `SKILL.md` and supporting scripts. Install any skill directly via the ClawHub CLI:

```bash
clawhub install <slug>
```

---

## Skills

| Skill | Slug | Description |
|-------|------|-------------|
| [ebay-agent](./ebay-agent) | `ebay-agent` | Search eBay for deals, estimate item value, rank by price/condition/seller trust. Requires free eBay developer credentials. |
| [domain-name-checker](./domain-name-checker) | `domain-name-checker` | Check domain availability across TLDs, get alternatives, brainstorm names with AI |
| [stock-quote](./stock-quote) | `stock-quote` | Get real-time stock quotes and basic financials. No API key required. |
| [cron-helper](./cron-helper) | `cron-helper` | Build and validate cron expressions in natural language. |
