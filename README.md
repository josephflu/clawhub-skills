# EagerBots Skills for ClawHub

A collection of OpenClaw skills published by [EagerBots](https://eagerbots.ai) on [ClawHub](https://clawhub.ai).

Each skill is a self-contained directory with a `SKILL.md` and supporting scripts. Install any skill directly via the ClawHub CLI:

```bash
clawhub install <slug>
```
